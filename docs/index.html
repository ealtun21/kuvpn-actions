Built on OpenConnect

KUVPN uses OpenConnect — the battle-tested, widely-audited open-source VPN client trusted by millions — to establish the actual tunnel. KUVPN handles the Azure AD authentication; OpenConnect does the rest.

Fully Open Source

Every line of KUVPN is public on GitHub under the MIT license. Inspect it, fork it, contribute to it — no black boxes.

Automated Azure MFA

Headless browser handles the Azure AD login and MFA flow automatically — no manual cookie hunting.

Auto Reconnect

Detects tunnel drops (not just the process) and reconnects up to 3 times with smart delays.

Session Persistence

Stays logged in across reconnects so you never have to re-authenticate mid-session.

Full Tunnel & Custom Script

Full tunnel routes all traffic through the university VPN. Power users can supply their own vpnc-script for full control.

Connection History

View past sessions with durations, timestamps, and reconnect counts — all stored locally.

Rolling Log File

Persistent 10 MB rotating log for post-mortem debugging — no lost context after a crash.
